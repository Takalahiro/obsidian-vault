# My Second Brain — 独创性梳理与完整技术报告

> 基准信息：版本 1.3.0｜最后更新 2026-05-20｜架构 Astro 6 SSG + Svelte 5 Islands｜无自建后端｜部署 Cloudflare Pages

---

## 第一部分：独创内容梳理

独创性分三级界定：
- **A 级（架构级独创）**：设计思想 + 系统抽象 + 取舍均为自己原创，业界罕见同类实现。
- **B 级（组合级独创）**：底层库为开源，但组合方式、工程编排、面向场景的封装是自己的。
- **C 级（标准集成）**：成熟方案的规范使用，构成工作量但不计为独创。

---

### A 级 — 架构级独创（共 7 项）

**A1. 三模互斥壁纸状态机（video / poster / ply）**
文档依据：`wallpaper-mode.ts`，三模通过 `useVideo` / `usePly` 布尔组合互斥，经 `patchFromMode()` 写回 localStorage；`BackgroundLayer` 用 `{#key scene.ply}` 强制 remount。
独创点：把 2D 视频、静态海报、3D 高斯点云收敛进单一状态键并保证互斥与无残留，是自定义的系统抽象。

**A2. 3D Gaussian Splatting 作为「不可交互全屏环境壁纸」的产品化用法**
文档依据：第 5 节。关闭 OrbitControls、`pointer-events:none`、`progressiveLoad:false` 全量加载后显示、常量机位 + 每帧 `lockCamera`、`integerBasedSort:false`。
独创点：3DGS 主流用于可漫游展示器，本项目反其道把它降格为背景层并配套相机锁定与 CSS 后期解耦，属于罕见用法。

**A3. HUD Canvas 实时壁纸引擎**
文档依据：`hud-wallpaper-engine.ts`，多层合成星场、星云、流星、视差、glitch；配套 `HudMissionStatus`（T+ / 信号格 / RA·DEC 每秒刷新）。
独创点：自研多层 Canvas 渲染引擎与任务 chrome，等同一个微型背景渲染引擎。

**A4. localStorage 多键分治状态架构**
文档依据：`second-brain:widgets` 与 `second-brain:notes-tools` 明确「不共享」；同一计算器在桌面侧键名为 `calculator`，在笔记侧为 `notes-tools`，并各自带 `*-layout` 键。
独创点：用命名空间分治解决「同一工具在桌面 vs 笔记内」的状态冲突，是前端状态管理的原创抽象。

**A5. 多 GPU / Worker 资源仲裁策略**
文档依据：第 7 节。Tab 切换释放 TF.js、`tf.dispose()` 离开 CNN 视图、Formula tab 隐藏时 Worker terminate、`document.hidden` 时暂停壁纸 rAF、Lazy import。
独创点：在同一 SPA 内让 TF.js / Transformers.js / Pyodide / 3DGS 四种重资源共存而不耗尽移动端 Safari WebGL context，仲裁策略是自己设计的。

**A6. 3DGS 资产体积绕过 Cloudflare 25 MiB 限制的双路径方案**
文档依据：PLY ~63 MiB 不入库（`.gitignore`），SOG ~10–11 MiB 入库并在 manifest 索引；运行时 `splatAssetUrl()` 把 `.sog` 映射回 `.ply`；`pnpm check:25mib` 审计。
独创点：manifest 索引 SOG、运行时映射 PLY、部署期手动上传的工程路径是原创取舍。

**A7. HUD 双层菜单栏（22px 状态条 + 44px 原菜单）渐进增强**
文档依据：`MacMenuBar.svelte` 双层布局，导航、语言、主题、时钟全部保留；小组件功能不被 HUD 隐藏，仅叠加视觉。
独创点：在不破坏既有导航 DOM 的前提下叠加 HUD chrome，是原创的渐进式增强实现。

附：**NotesToolHost 双触发修复**（v1.3.0：阻止 tile 级 pointer 冒泡避免一次点击触发两次 toggle）属于自己发现并解决的真实工程问题，归入 A4 的实现细节。

### B 级 — 组合级独创（共 6 项）

**B1. Pyodide + `sys.settrace` + AST 规则模板教学解释器**
文档明确：解释为「rule-based AST templates，not LLM-generated」。底层 Pyodide/ast 为开源，但把追踪事件转为可读步骤的模板是自写规则。

**B2. FormulaNet OCR + 双预处理 + latexQualityScore 选择**
文档：Web Worker 内 Transformers.js 跑 FormulaNet，双预处理（thicken / no thicken）后由 `latexQualityScore` 选优。质量评分启发式与 Worker 调度为自写工程。

**B3. 构建期数据流水线（4–5 个 JSON artifact）**
文档：`prepare:vault` 顺序产出 `notes-mtime.json` / `stats.json` / `wikilinks.json` / `media-manifest.json`，并在 CI 中 `git fetch --unshallow` 保证 mtime 准确。编排顺序与 git mtime 优先策略是自己的。

**B4. Remark/Rehype 自定义插件链**
文档：6 个插件链路，其中 `remark-obsidian-image.mjs`、`remark-normalize-math.mjs`、`remark-mermaid.mjs` 为自写插件（`.mjs`）。构建期 AST 转换、运行时零补丁。

**B5. 双链 slug 归一化 + 红/蓝 stub 区分**
文档：`slugify` + `build-wikilinks.mjs` 构建期解析 `[[links]]`，断链落入 `broken` / `orphans`，UI 红色提示。归一化与悬空/孤岛检测为自写。

**B6. MNIST LeNet 双路径预处理 + vizModel 中间层 3D 可视化**
文档：280×280 canvas 经「direct scale | crop+thicken」双路径，TF.js LeNet 推理 + `vizModel` 层输出，NetworkPanel 提供 SVG / 2D 特征图 / Three.js 3D。组合呈现方式为自写。

附：MATLAB 计算器的矩阵分步消元引擎（`src/lib/matrix/`）、逻辑真值表解析、统计假设检验分步，若算法为自写，可上升为 A 级；此处保守列入 B 级。

## 第二部分：完整技术报告

### 1. 项目定位
将 Obsidian vault 发布为静态 Web 知识库，并在首页叠加可配置「桌面 OS 壳」（壁纸、时钟、图谱、ML 实验室、计算器、白板等）。内容在构建期编译为静态 HTML，重交互模块在浏览器按需加载，无自建后端。

### 2. 系统分层
- UI 层：WidgetHost（首页）/ NotesToolHost（笔记页）/ 工具页
- 皮肤层：`data-ui` = mac | glass | pixel | hud
- 状态层：localStorage（widgets / notes-tools / ui-skin / 各 layout 键）
- 壁纸层：BackgroundLayer（video | poster | 3DGS | HUD），z-index: -1
- 构建期数据：wikilinks.json / stats.json / media-manifest.json / notes-mtime.json
- 内容源：obsidian-vault（git submodule）

### 3. 构建流水线（`prepare:vault`，dev/build 前置）
1) 校验/初始化 vault submodule；2) CI 中 `git fetch --unshallow`；3) 同步 vault 资源到 `public/vault-assets/`；4) 生成四个 JSON artifact。Astro Content Layer 经 Unified remark/rehype 链输出静态 HTML。

### 4. 客户端分包（Vite manualChunks）
- `widget-core`：wallpaper-mode、media（首页加载）
- `wallpaper-three`：Three.js、gs3-wallpaper（点云模式加载）
- 各工具页：Pyodide、TF.js、Cytoscape 等路由 lazy import

### 5. 关键子系统
- 三模互斥壁纸（A1）：详见第一部分。视频用 layerA/layerB 双缓冲 crossfade；移动端 UA 检测降级为 poster。
- 3DGS 环境壁纸（A2/A6）：主渲染器 `@mkkellogg/gaussian-splats-3d`；`spatial-camera.ts` 处理陀螺仪（clamp ±0.14）与极弱鼠标视差（POINTER_SENSITIVITY=0.004）；CSS 后期（径向羽化、0.3px 柔焦、暖调、1.5s 缓入）与 WebGL 解耦；`disposeChain` + `AbortController` + `visibilitychange` 管理生命周期；加载超时 120s。
- HUD 皮肤（A3/A7）：Canvas 实时壁纸引擎 + 双层任务 chrome + 日全食 intro + CLI 搜索样式（`MSB://CMD>`）。
- Obsidian 内容管道（B4/B5）：6 个 remark/rehype 插件，构建期 AST 转换。
- 浏览器内计算（B1/B2/B6）：Python IDE（Pyodide + settrace + AST）、Neural Lab（TF.js LeNet + FormulaNet OCR Worker + Pyodide/SymPy）、MATLAB 计算器（矩阵/微积分/离散/统计/表达式）。
- 资源仲裁（A5）：见第一部分。

### 6. 部署与约束
Cloudflare Pages，输出 `dist`，必须包含 git submodule。单文件 ≤ 25 MiB；FormulaNet（~77 MiB）运行时从 HF CDN 下载，SymPy 首次 ~10–20 MB 由 Pyodide CDN 加载，MNIST 权重 ~879 KB 入库。

### 7. 测试与验证
Playwright 点云冒烟（`test-ply-wallpaper.mjs`）、多场景对比、gs3 诊断、`check:25mib` 体积审计、`check:self` 路由自检。

### 8. 已知限制
PLY 首载 10–40s；移动端 3DGS 降级 poster；iOS 需授权陀螺仪；大 PLY 必须部署期上传不入 git。后续可选：KSplat 预处理、Service Worker 缓存、场景 LOD。
