## 大部分编程语言通用的结构和语法

编程语言虽然语法各异，但底层的编程思想和结构是相通的。以下是几乎所有主流编程语言都具备的核心概念：

### 1. 数据类型

**基本数据类型**（几乎所有语言都有）：
- **整数类型**：`int`, `long`, `short` 等
- **浮点类型**：`float`, `double` 等
- **布尔类型**：`boolean`, `bool` 等（true/false）
- **字符/字符串类型**：`char`, `string` 等

```java
// Java
int age = 25;
double price = 99.99;
boolean isValid = true;
String name = "张三";
```

```python
# Python
age = 25
price = 99.99
is_valid = True
name = "张三"
```

```javascript
// JavaScript
let age = 25;
let price = 99.99;
let isValid = true;
let name = "张三";
```

```cpp
// C++
int age = 25;
double price = 99.99;
bool isValid = true;
string name = "张三";
```

### 2. 变量声明与赋值

所有语言都支持变量的声明和赋值，只是语法略有不同。

```java
// Java（静态类型，需要声明类型）
int x = 10;
x = 20;
```

```python
# Python（动态类型，不需要声明类型）
x = 10
x = 20
```

```javascript
// JavaScript（动态类型）
let x = 10;
x = 20;
```

```go
// Go
var x int = 10
x = 20
// 或简写
x := 10
```

### 3. 运算符

**算术运算符**：`+`, `-`, `*`, `/`, `%`  
**比较运算符**：`==`, `!=`, `>`, `<`, `>=`, `<=`  
**逻辑运算符**：`&&` (AND), `||` (OR), `!` (NOT)  
**赋值运算符**：`=`, `+=`, `-=`, `*=`, `/=`

```java
// Java
int a = 10, b = 3;
int sum = a + b;        // 13
int mod = a % b;        // 1
boolean result = a > b; // true
```

```python
# Python
a, b = 10, 3
sum = a + b        # 13
mod = a % b        # 1
result = a > b     # True
```

```javascript
// JavaScript
let a = 10, b = 3;
let sum = a + b;        // 13
let mod = a % b;        // 1
let result = a > b;     // true
```

### 4. 条件语句（分支结构）

**if-else 语句**是所有语言的标配。

```java
// Java
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}
```

```python
# Python（使用缩进代替大括号）
if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

```javascript
// JavaScript
if (score >= 90) {
    console.log("优秀");
} else if (score >= 60) {
    console.log("及格");
} else {
    console.log("不及格");
}
```

```cpp
// C++
if (score >= 90) {
    cout << "优秀" << endl;
} else if (score >= 60) {
    cout << "及格" << endl;
} else {
    cout << "不及格" << endl;
}
```

**switch-case 语句**（部分语言支持）：

```java
// Java
switch (day) {
    case 1:
        System.out.println("星期一");
        break;
    case 2:
        System.out.println("星期二");
        break;
    default:
        System.out.println("其他");
}
```

```javascript
// JavaScript
switch (day) {
    case 1:
        console.log("星期一");
        break;
    case 2:
        console.log("星期二");
        break;
    default:
        console.log("其他");
}
```

```python
# Python 3.10+ 支持 match-case
match day:
    case 1:
        print("星期一")
    case 2:
        print("星期二")
    case _:
        print("其他")
```

### 11. 面向对象（类与对象）

大多数现代语言都支持面向对象编程。

```java
// Java
class Person {
    String name;
    int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public void sayHello() {
        System.out.println("我是" + name);
    }
}

Person p = new Person("张三", 20);
p.sayHello();
```

```python
# Python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def say_hello(self):
        print(f"我是{self.name}")

p = Person("张三", 20)
p.say_hello()
```

```javascript
// JavaScript
class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    sayHello() {
        console.log(`我是${this.name}`);
    }
}

let p = new Person("张三", 20);
p.sayHello();
```

### 12. 异常处理

```java

    int result = 10 / 0;
```

```python
    result = 10 / 0
```

```javascript

    let result = 10 / 0;

    console.log("总是执行");
```

### 15. 学习建议

**掌握一门语言后，学习其他语言会很快**，因为：

1. **核心概念相同**：变量、循环、函数、面向对象等
2. **只需学习语法差异**：如 Python 用缩进，Java 用大括号
3. **编程思维是通用的**：算法、数据结构、设计模式

**学习路径**：
- 先深入学习一门语言（如 Java）
- 掌握编程思想和算法
- 再学其他语言时，重点关注语法差异和特性

**示例：同一个问题的多语言实现**

```java
// Java：计算数组和
public static int sum(int[] arr) {
    int total = 0;
    for (int num : arr) {
        total += num;
    }
    return total;
}
```

```python
# Python：计算列表和
def sum_list(arr):
    total = 0
    for num in arr:
        total += num
    return total

# 或使用内置函数
total = sum(arr)
```

```javascript
// JavaScript：计算数组和
function sum(arr) {
    let total = 0;
    for (let num of arr) {
        total += num;
    }
    return total;
}

// 或使用 reduce
let total = arr.reduce((a, b) => a + b, 0);
```

**结论**：编程的核心是**思想和逻辑**，语法只是工具。掌握了一门语言的核心概念，学习其他语言就是"换个语法写同样的逻辑"。
